Веб-приложение `momentum`, аналог [расширения](https://chromewebstore.google.com/detail/momentum/laookkfknpbbblfpciffpaejjkokdgca?hl=ru&pli=1).
В приложении есть:
- часы
- дата
- динамичный фон, который зависит от времени суток
- виджет погоды
- список задач
